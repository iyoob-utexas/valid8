# Test Grid

This markdown file is the text-based equivalent of the Excel test grid.
It defines the baseline checklist for data testing, including tier, category, rule,
quality dimension, tooling guidance, failure action, and DPPF test ID mappings.

The DPPF ID column links each row to the corresponding adversarial test definition in `docs/tests/`. Multiple IDs indicate that a single grid row covers more than one DPPF test category.
